# Система постановки задач "Поток"

В директории "imitation_modelling" находится система имитационного моделирования поведения реальных 
обработчиков задач в распределенных системах.  

В директории "service" находится реализация системы постановки задач.  

На данный момент реализована только система имитационного моделирования для выбора оптимального алгоритма 
балансировки нагрузки.
